Welcome to SCAJA official website

This website is a job finder application that takes into consideration context information (education, specialization, job interests, experience, etc) related to the user, which are retrieved from the social network LinkedIn.

The app provides the capability to the user to login with LinkedIn's account, search and automatically receive personalised recommendations on job postings. On the other hand the application will allow employers to login so as to browse through job seekers that have posted their resume and find candidates that could be suitable for the position.
